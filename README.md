# Openbyte

This repository contains the source code of the official Openbyte website. The project is designed to be built and executed in a local development environment using standard Node.js tooling.

## Build and Execution Process

To compile and run the project locally, follow the steps outlined below:

### 1.Clone the repository
Obtain a local copy of the source code by cloning the official Openbyte repository:
```bash
git clone https://github.com/openbytelabs/openbyte
```

### 2. Navigate to the project directory
Change the working directory to the root of the project:
```nash
cd openbyte
```

### 3. Install dependencies
Install all required project dependencies using the Node Package Manager (npm):
```bash
npm install
```

### 4. Run the development server
Start the local development server to build and serve the application:
```bash
npm run serve
```

***Once these steps are completed successfully, the Openbyte website will be available for local development and testing.***
